# pebble-tool-pkgbuild
Install Pebble's cli tool on Arch Linux. Used to install their SDK, build apps/watchfaces, and test apps/watchfaces in an emulator, etc. 


Steps to use
1. Install this PKGBUILD (e.g., download the file and run `makepkg -si` in the same directory)
2. Follow pebble's official documentation for building and testing apps/watchfaces https://developer.repebble.com/sdk/#create-a-pebble-app
